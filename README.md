### 페이지 별 사용 기능 🎅🏻
| 메뉴  | 사용 기능 |
| ------------- |:-------------:|
| 홈      | 1. 인기 종목 (useState)     |
| 관심목록      | 1. 종목 데이터 (props)<br /> 2. 정렬 옵션(useState)     |
| 종목검색      | 1. 종목 데이터 (json)<br /> 2. 검색(useState)     |
| 토론      | 1. 공감/비공감 버튼, 전체 댓글 개수 (useState)<br /> 2. 댓글 수정(Javascript DOM) |
